# PS4 BORDER BREAK アセンシミュレーター

PS4版「BORDER BREAK」の機体パーツ、武器、チップ、要請兵器を選択して、積載猶予や主要ステータスを確認するためのアセンシミュレーターです。

## できること

- 頭、胴、腕、脚パーツの選択
- 強襲、重火力、遊撃、支援の武器選択
- 強化段階の切り替え
- チップ選択とステータス反映
- 要請兵器、サテライトバンカーの積載反映切り替え
- 積載超過時の歩行、ダッシュ、巡航への影響表示
- 選択中アセンのコピー
- PC、スマホ向けのレスポンシブ表示

## 使い方

公開ページをブラウザで開くと使えます。

## データについて

パーツ、武器、チップ、要請兵器のデータは、PS4版BORDER BREAKのWikiから保存したHTMLを元に変換しています。

元データの取得、変換に使った補助スクリプトも `outputs` フォルダ内に残しています。

- `scrape_borderbreak_wiki.py`
- `convert_saved_wiki_html.py`
- `data.json`

通常のWeb公開には `data.json` やPythonスクリプトは必須ではありません。

## 注意

個人制作の非公式ツールです。

ゲーム内の実数値、Wikiの記載、変換処理の都合により、表示内容に誤差や未対応項目が含まれる可能性があります。
